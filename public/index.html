Can You Beat the AI?

The following is a clone to the famous flappy bird game which plays purely by machine learning.

How it works

The following AI implements the genetic algorithm which simulates biology ---- take the best bird that got the most score and make the next generation with the "genes" of this best bird. Eventually the bird will out smart the game and get to, well, infinity. Try out the buttons and see it for your self.
